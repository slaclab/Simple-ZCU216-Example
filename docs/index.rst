Simple-ZCU216-Example Documentation
=====================================

Reference firmware and software example for the Xilinx ZCU216 evaluation
board (Xilinx Zynq UltraScale+ RFSoC, FPGA part
``xczu49dr-ffvf1760-2-e``). Built on the SLAC ``axi-soc-ultra-plus-core``
platform; shared workflow documentation lives on the
:hub:`platform docs site <index.html>`.
